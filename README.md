# 🖥️ Applications-On-Windows-Forms

**Applications-On-Windows-Forms** is a C# solution built using **Windows Forms (WinForms)** to demonstrate simple graphical desktop applications.
The repository focuses on building interactive UI-based programs with event-driven programming in a beginner-friendly way.

---

## 📦 Projects Included

### 1. 🍕 Pizza Cashier (UI Only)

A simple Windows Forms interface that simulates a pizza ordering cashier system.

**Features:**

* Clean and interactive user interface
* Menu layout for pizza selection
* Order form design (UI demonstration only)
* Basic input controls (buttons, checkboxes, labels)

**Purpose:**

* Practice designing Windows Forms layouts
* Learn UI component arrangement
* Understand event-driven UI structure

---

### 2. ❌⭕ Tic-Tac-Toe Game

A classic two-player Tic-Tac-Toe game built using Windows Forms.

**Features:**

* Two-player gameplay (X vs O)
* Interactive button-based grid
* Win/draw detection logic
* Game reset functionality

**Concepts Used:**

* Event-driven programming
* Game logic implementation
* UI state management
* Conditional logic and loops

---

## 🚀 Getting Started

### Prerequisites

* Visual Studio (recommended)
* .NET Framework or .NET (WinForms support)
* Windows OS

---

### Running the Projects

1. Clone the repository:

```bash id="t7p2av"
git clone https://github.com/your-username/Applications-On-Windows-Forms.git
```

2. Open the solution file in **Visual Studio**

3. Select the desired project:

   * Pizza-Cashier (UI demo)
   * Tic-Tac-Toe (game)

4. Click **Run ▶**
